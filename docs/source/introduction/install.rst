Installing labcontrol
=====================

If you want to try labcontrol, you first have to decide on what you want to use labcontrol for:

1. As an user, not having any Python programming skills. Your aim is to use labcontrols measurement scripts and Jupyter notebooks "as-is". 
With the exception of simpel edits, you have absolutely no intention to add nor to change any functionality. To be able to make those simpel
edit, you'll probably only need labcontrols apidoc and some sort of simple texteditor. 
2. As a developer. You also want to use labcontrol, but being a (Python) developer, you will probably rather check out the code yourself in 
your preferred IDE, before starting to try to add new functionality.

This install manual will only focus on the first type of user, because an developer wants to have full control the Python environment used.
